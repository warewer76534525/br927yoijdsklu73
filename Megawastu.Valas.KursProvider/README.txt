1. melakukan instalasi dotnet framework 4.0
2. membuat folder pre requsite utk win service excel

For 64-bit (x64), create this folder:
C:\Windows\SysWOW64\config\systemprofile\Desktop

For 32-bit (x86), create this folder:
C:\Windows\System32\config\systemprofile\Desktop

3. melakukan instalasi windows service dengan mengeksekusi build install.service

4. start dari control panel - services
